What's new in ADK4.4?

TWS Mono

True wireless(TWS) mono decoding and relay feature in hydracore and Bluecore based ADK sink application. Whenever the system is in a TWS mono scenario, the master renders one channel and forwards other channel to peer device. That would improves power consumption and long lasts battery and some more MIPS is available at DSP/audio subsystem.Further details are described in ADK Audio Sink Application Peer Device User Guide

Audio to Video synchronization & Synchronization between audio outputs

Audio to Video synchronization improvements in TWS. This needs explicitly enabling certain compiler flags in DSP projects. Further details are described in ADK Audio Sink Application Peer Device User Guide

OTAU Upgrade optimizations

Optimized OTA DFU transfer rates over BLE. The OTA time required for DFU is drastically improved in this release.
