Detailed Description

[SDK Folder]/src/kalimba/spdif/firmware_interface.asm

DESCRIPTION

DSP receives messages from firmware about the status of the spdif Rx stream. Vm needs to enable this feature in order to firmware sending DSP the releavant messages.

Messages that are processed by DSP are: $MESSAGE_SPDIF_CHNL_STS_EVENT: channel status words for spdif Rx stream $MESSAGE_AUDIO_STREAM_RATE_EVENT: Indicates the rate of audio stream $MESSAGE_SPDIF_BLOCK_START_EVENT: Indicates that the block start has been received
