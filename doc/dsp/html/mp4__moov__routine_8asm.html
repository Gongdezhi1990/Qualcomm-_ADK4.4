$aacdec read_uintvar4 ( )

aacdec.read_uintvar4 reads a uintvar of upto size 4 from the bitstream only upto 23 are actually read and it is assumed that the value of the uint is smaller than 2^23 - 1, otherwise saturates - this really shouldn't happen though.
