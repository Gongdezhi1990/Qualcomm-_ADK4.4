Function Documentation

$math biquad ( )

Library subroutine for a bi-quad IIR filter. Equation of each section:

Parameters
INPUTSI0 = pointer to the input data L0 = length of input buffer (it will be circular) I4 = pointer to the output data L4 = length of output buffer (it will be circular) I1 = scaled coefficients b2,b1,b0,b_scale, a2,a1,a_scale,... etc L1 = 7 * num_biquads I5 = delay line L5 = 2 * num_biquads + 2 r0 = no of samples to apply the filter to r10 = number of biquad sections
OUTPUTSI0 = updated over data read I4 = updated over data written
Trashed Registers:
r0-r5, r10, DoLoop
  y(n) =   (b0*x(n) + b1*x(n-1) + b2*x(n-2)) << b_scale
         - (          a1*y(n-1) - a2*y(n-2)) << a_scale

The implementation uses Direct Form I as shown in:

http://en.wikipedia.org/wiki/Digital_biquad_filter

The "b" coefficients can be independently scaled from the "a" coefficients to make optimal use of the available bits for precision.
