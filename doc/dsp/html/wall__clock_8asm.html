Detailed Description

Provides a function to be able to view a time (in us) of a 'wall clock' that is synchronised between 2 BT devices.

Part of ADK_CSR867x.WIN. 4.4

Function Documentation

$wall_clock disable ( )

Disables the 'wall clock' on a particular BT link.

Parameters
INPUTS
  • r1 = pointer to a variable that stores the 'wall clock' structure, should be of length $wall_clock.STRUC_SIZE
OUTPUTS
  • none
Trashed Registers:
r0-5, r10, DoLoop
Notes:

This should be called once the 'wall clock' is not needed anymore on a particular link

$wall_clock enable ( )

Enables the 'wall clock' on a particular BT link.

Parameters
INPUTS
  • r1 = pointer to a variable that stores the 'wall clock' structure, should be of length $wall_clock.STRUC_SIZE
OUTPUTS
  • none
Trashed Registers:
r0-5, r10, DoLoop
Notes:

This must be called before a call to $wall_clock.get_time. The structure should be filled out with the BT address prior to the call: 

M[r1 + $wall_clock.BT_ADDR_WORD0_FIELD] = 16bit NAP address
M[r1 + $wall_clock.BT_ADDR_WORD1_FIELD] = UAP in MS byte,
                                          MS byte of LAP in LS byte
M[r1 + $wall_clock.BT_ADDR_WORD2_FIELD] = LS 16bits of LAP address

$wall_clock get_time ( )

Get the 'wall clock' time on a particular BT link.

Parameters
INPUTS
  • r1 = pointer to a variable that stores the 'wall clock' structure, should be of length $wall_clock.STRUC_SIZE
OUTPUTS
  • r1 = 'wall clock' time
Trashed Registers:
none
Notes:

Before calling this function for the first time the $wall_clock.init_time function should be run.

$wall_clock message_handler_failed ( )

Message handler for $MESSAGE_WALL_CLOCK_FAILED and $MESSAGE_WALL_CLOCK_FAILED_TBTA messages.

Parameters
INPUTS
  • r0 = message ID $MESSAGE_WALL_CLOCK_FAILED:
    • r1 = message Data 0 (BT address: NAP)
    • r2 = message Data 1 (BT address: UAP (in MS 8 bits) + MS byte of LAP)
    • r3 = message Data 2 (BT address: LS word of LAP) $MESSAGE_WALL_CLOCK_FAILED_TBTA:
    • r1 = message Data 0 (BT address: Type)
    • r2 = message Data 0 (BT address: NAP)
    • r3 = message Data 1 (BT address: UAP (in MS 8 bits) + MS byte of LAP)
    • r4 = message Data 2 (BT address: LS word of LAP)
OUTPUTS
  • none
Trashed Registers:
r0, r1, r2
