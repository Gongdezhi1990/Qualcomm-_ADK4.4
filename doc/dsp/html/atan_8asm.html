Function Documentation

$math atan ( )

Library subroutine to evaluate the arctan (4 quadrants) of a complex number. The approximation uses a second order polynomial, optimized with a minimax criterion (see IEEE Signal Processing Magazine, Volume 23 Number 3 May 2006. The maximum absolute error is 0.0038 rad.

Parameters
INPUTS
  • r5: Real part
  • r6: Imaginary part
OUTPUTS
  • r5 = angle -2^23 == -180°, +2^23 == 180°
Trashed Registers:
rMAC, r0, r1, r2, r3, r4, r5, r6, r7, r8

PROGRAM MEMORY: 51 words

CPU USAGE:

44 cycles for worst case scenario.
