Function Documentation

$mp3enc find_boundaries ( )

finds the region boundaries in a granule, directly modifies $mp3enc. bigvaluesx2 : number of samples in big_value region rzero : number of samples in rzero region count1 : number of samples in count1 region region0_count : 1 less than the number of scf bands in region 0 region1_count : 1 less than the number of scf bands in region 1 region_end[3] : the index of last sample in each region

MODIFICATIONS:

Parameters
INPUTS
  • r9 = $mp3enc.memory_pointer (remains unchanged)
OUTPUTS
  • none
Trashed Registers:
- r0-r8, r10, rMAC, I0-I3, I5-I7, M0, M2-3
$mp3enc find_boundaries optimal_partition ( )
  • finds the optimal combination of number of subbands in regions 0, 1 and 2
Parameters
INPUTS
  • M0 = 1
  • r5 = grch
  • r6 = bigval_sbfs = number of bigval subbands minus 1
  • r9 = $mp3enc.memory_pointer (remains unchanged)
OUTPUTS
  • updates M[&$mp3enc.region0_count + r5] and M[&$mp3enc.region1_count + r5]
  • r4 = M[$mp3enc.bigvaluesx2 + r5] (see notes)
Trashed Registers:
- r0-3, r7-8, I0-3 I5-7, M2-3, rMAC, r10
  • also $scratch.s0 and $scratch.s1 in external mode
Notes:

- r4 and r5 are trashed and then restored
