$cos_taylor6 ( )

Library subroutine to evaluate the Cosine of a number.

Parameters
INPUTSr0 = input angle -2^23 == -180, +2^23 == 180
OUTPUTSr0 = result (accurate to approximately 15 bits for 6th order)
Trashed Registers:
I0, r1, r2, r3, rMAC
CPU USAGE:
TBD
Notes:

More efficient implementation needs to be done after it is determined how many bits of precision are needed in cosine function Then an efficient trade off needs to be made between table size and order of taylor interpolation done between table entries. This implementation is a straight taylor series implementation of order 6 with no table used and its centered around pi/4. It makes use of symmetry cos x = cos (- x) and cos(pi - x) = - cos(x)
