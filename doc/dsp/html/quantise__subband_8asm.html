Function Documentation

$mp3enc quantise_subband ( )

quantises all the samples in a sub-band, and obtains estimates of quantisation noise energy and number of bits needed for coding the subband samples

Parameters
INPUTS
  • I0 = start of the to-be-quantised values
  • I4 = put the quantised values here
  • r0 = number of samples
  • r1 = Nx4
  • M0 = 1
  • M3 = 9;
  • M2 = 32;
  • I1 = &$mp3enc.requant_cnt;
  • I6 = (&$mp3enc.x43_lookup2 - 9);
OUTPUTS
  • I4 = I4 + r0
  • I0 = I0 + r0
  • r0 = unchanged
  • r1 = estimated number of bits
  • r5 = quantisation noise energy = sum(n^2)
  • r8 = OR of all quantised samples, if 0 all samples are zero
Trashed Registers:
- r2-8, r10, rMAC, I2, I7, M1 (NOT I5, I3)
  • in case of MP3ENC_USE_EXTERNAL_MEMORY defined: $scratch.s0, $scratch.s1
