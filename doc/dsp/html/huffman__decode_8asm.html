$aacdec huffman_getbits ( )

Get bits from input buffer that can be called when we're in the middle of reading huffman data. It's needed for reading noise_pcm scalefactors.

Parameters
INPUTS
  • I0,L0 = buffer to read words from
  • r5 = number of bits to get from buffer
  • r2 = current word from bitstream
  • r3 = current bitmask to use to read from bitstream
  • M1 = 1
OUTPUTS
  • I0 = updated
  • r1 = the data read from the buffer
  • r2 = updated
  • r3 = updated

- r5
