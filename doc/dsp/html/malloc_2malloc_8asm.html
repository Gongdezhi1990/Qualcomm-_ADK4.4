Function Documentation

$malloc get_block_info ( )

Returns the address of the structure in the info block for a block of RAM. It does not check if the address is the start of a block.

Parameters
INPUTSr0 = address
OUTPUTSI3 = address of $malloc.POINTER_DM1_FIELD or $malloc.POINTER_DM2_FIELD element of info structure (whichever is appropriate for the address input) r0 = 'actual' start address of block
Trashed Registers:
r1

rMAC, Div (if MALLOC_FREE_TESTS defined)

$malloc get_block_size ( )

Returns the size of a block of RAM, does not check if the address is the start of a block.

Parameters
INPUTSr0 = address
OUTPUTSr0 = size (0 on failure)
Trashed Registers:
r1, I3, M0

rMAC, Div (if MALLOC_FREE_TESTS defined)

$malloc realloc_pools ( )

Reallocates the memory pools. Pass in a block with the new set of memory pools (a list of $malloc.input structures). This must be the only block that is currently allocated.

Parameters
INPUTSr0 = DM1 area pointer (NOTE: THIS SHOULD NOT BE ZERO) r1 = DM1 area size r2 = DM2 area pointer r3 = DM2 area size r4 = block with new pool information
OUTPUTSr0 = result, 0=success, 1=fail
Trashed Registers:
Assume everything
$malloc zero_alloc ( )

Allocate a block and zero all the elements of it.

Parameters
INPUTSr0 = size r1 = dm preference (0x1: dm1, 0x2: dm2, 0x3: either)
OUTPUTSr0 = returned address (0 is failure) r1 = actual size of block
Trashed Registers:
r10, I3, M0
