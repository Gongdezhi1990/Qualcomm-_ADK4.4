Write an arbitrary block of data to a log buffer with a message id/length header. Can be called from a C function.

Parameters
INPUTS
  • r0 = 8 bit message id
  • r1 = pointer to the message
  • r2 = length of the message in words
OUTPUTS
  • none
Trashed Registers:
r1, r2, r3, r10, I4, I0, L0
