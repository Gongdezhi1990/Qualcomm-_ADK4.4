or $math.fsqrt - single calling function (float format input) or $math.sqrt.init $math.sqrt.core - loop calling (double precision input) $math.fsqrt.core - loop calling (float format input)

Parameters
INPUTS
  • rMAC = x (double presicion, Q1.47 positive number) (ie. 0 and 1.0 fractional)

  • r0 = y (Q1.23) (accurate to 16bits, RMS error is 5bits))

  • rMAC = y (mantisa)
  • r1 = y (exp: negative number of sign bits)
