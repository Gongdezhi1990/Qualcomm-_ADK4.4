Function Documentation

$mp3enc analysis_filterbank ( )

This analysis subband filterbank is used to split the broadband signal with sampling frequency fs into 32 equally spaced subbands with sampling frequencies fs/32.

The steps are:

  • Input 32 audio samples
  • Build and input sample vector X of 512 elements. The 32 audio samples are shifted in at positions 0 to 31, the most recent one at position 0 and the 32 oldest elements are shifted out. (Here done for 'free' by using a circular buffer)
  • Window vector X by vector C C can be found in table C.1 in the MPEG1 standard (ISO/IEC 11172-3)
  • Calculate Yi from X Yi = Sum(j=0:7)of (Z(i+64j)) for i=0:63
  • Calculate the 32 subband samples Si by matrixing Si = Sum(k=0:63) of (Mik * Yk) for i = 0:31 where Mik = cos((i+0.5)*pi/32*k) (Here, a shorter method is used based on redundancy in M)

causes about 828 stalls per call (i.e. about 5% of the used MIPS)

MODIFICATIONS:

Parameters
INPUTS
  • I0 = pointer to first element of output array, length 576
  • I1 = pointer to front of shift_input array (CIRCULAR), length 512
  • I4 = pointer to first element of input array, length 576
  • L4 = length of input array = 576
  • L1 = length of shift_input array (512)
  • r9 = $mp3enc.memory_pointer (remains unchanged)
OUTPUTS
  • I1 = pointer to front of shift_input array (CIRCULAR), this will be 64 behind I1 input, length 512
Trashed Registers:
- r0-4, r6, r8, r10, I0, I2-5, M0-3 (NOT r5)
