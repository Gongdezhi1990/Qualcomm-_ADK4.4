$mp3enc mdct windowed_mdct ( )

inital set up to do single 36point mdct or three 12point mdcts. mdct performed by another module and this organises the results to be sent back to calling module.

MODIFICATIONS:

Parameters
INPUTS
  • I4 = pointer to first element of current 18 element section of 'previous' array
  • I5 = pointer to first element of current 18 element section of 'current' array
  • r0 = block window type 0 - normal 1 - start 2 - short 3 - stop
  • M2 = 3
  • M1 = 1
  • M0 = 17
OUTPUTS
  • none
