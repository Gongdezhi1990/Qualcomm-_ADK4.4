Function Documentation

$mp3enc put_res_bits ( )

Writes bits to reservoir.

Parameters
INPUTS
  • r0 = number of bits to put in buffer
  • r1 = the data to put in the buffer
  • I0 = circular buffer pointer to write words to
  • r7 = number of remaining bits in intermediate register buffer (put_res_bitpos)
  • r8 = intermediate register buffer (put_res_nextword)
  • M0 = 1
  • L0 = size of reservoir
OUTPUTS
  • I0 = circular buffer pointer to write words to (updated)
  • r7 = number of remaining bits in intermediate register buffer updated
  • r8 = intermediate register buffer updated
Trashed Registers:
- r2
Notes:

- this is written assuming RES_BITS_PER_WORD = 24, otherwise the the intermediate register needs to be masked accordingly

  • IMPORTANT: all bits in r1 should be zero except the less significant bits determined by r0
